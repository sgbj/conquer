# Conquer

A Conquer Online private server for patch 4330. Built with C#, ASP.NET Core, and Entity Framework Core.

## Features

* 💬 Commands
* 🗡️ Items
* 🤖 NPCs
* 💰 Shops
* 🧌 Monsters
* 👑 Guilds
* ⚔️ Attacks
